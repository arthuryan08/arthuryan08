## 🚀 What I'm Building

<table>
<tr>
<td width="50%" valign="top">

### 🔷 Klozr
**B2B SaaS — Sales Prospecting for Brazil**

Next.js 15 · TypeScript · Supabase · Claude API · BullMQ

A platform that unifies **search → enrichment → outreach → tracking** for Brazilian SMBs.

**Key features shipped:**
- Company search via CNAE + Municipality (Federal Revenue data)
- AI-powered personalized message generation
- Kanban pipeline with drag-and-drop
- Anti-ban system for WhatsApp mass outreach
- CSV import with automatic data enrichment

</td>

**RAG, Semantic Search & Document Processing**

Python · Qdrant · pgvector · n8n · LLM APIs

Building AI-powered systems for corporate and government clients:

- RAG pipelines for document analysis
- Semantic search with vector databases
- Legal norms analysis for public administration
- Multi-tenant SaaS platforms (marketplaces, ERPs, dashboards)
- Automated essay correction with Claude API
